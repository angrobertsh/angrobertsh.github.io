The Writer's Toolkit
Instructions here. Tutorial video here.
The Writer's Toolkit is a web application that visualizes the emotional word content in a piece of text. It runs right in your browser (but not Internet Explorer [sorry]) and it works on plain text of almost any length. I created this app with jQuery, d3.js, and html for his 2012 MSc project under the supervision of Jon Oberlander, Micha Elsner, and Victoria Adams at the University of Edinburgh.
Illustrator Layouts
I've created a set of layouts for a variety of contexts with Adobe Illustrator to show how I would implement different visual aesthetics for things like magazine layouts, app interfaces, content websites, and posters. Reddit Visual Redesign
For my beginning web design class, I was asked to make a visual redesign of reddit.com. I took to the project with gusto and generated an html/css/javascript visual layout that has functioning elements (like hover effects and drop down bars, ooo).
